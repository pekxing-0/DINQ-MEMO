03 · 打法 · 三步

不是平行业务
是 演化路径

每一步都为下一步攒"训练集 + 分发通路 + 客户成功证据"。

1

MCP 通道层

M1–M6 · land grab

dinq-mcp 进 Claude Code / Cursor / 企业内部 Agent · 三档定价：Free / Pro $99 / Enterprise · 目标：成为"AI 招聘场景下事实标准 MCP"。

2

Tier 1/2/3 梯队规模化 + 多形态交付

M3–M18 · 收入主力

C 端 DINQ Card 2.0 + B 端年签三档（30 万 / 80–120 万 / 200–400 万）· 交付六形态：字库 + 上岗 + Work Trial + 论文分享 + 内部沙龙 + 战略咨询。

3

Vertical Hiring Agent

M12+ · 终局

DINQ Agent 自主跑：识别 → 匹配 → Work Trial → 交付。按 outcome 收费、毛利 80%。对标 Mercor 路径，但不雇佣——做识别 + 介绍 + 验证。

关键

没有 MCP 阶段的高频调用 + 意图信号 + C 端作品池 + B 端真实需求，第三步 Agent 化就是空中楼阁。三步之间是数据飞轮关系。

DINQ

识别 → 匹配 → 交付 · outcome-based · 终局

最关键咬合点

MCP 频次 ↔ B 端年签。MCP 让企业习惯用 DINQ 思考人才问题——习惯到阈值，年签是自然结果。MCP 是养出年签的，不是直接卖 MCP 赚钱。

终局

① + ② + ③ + ④ 累积的数据 = 训练 ⑤ Vertical Hiring Agent 的核心语料。

10 · 今晚通话 · 三件必戳的事

把这三件事
给 Sam 戳到底

①

MCP 是通道、不是终点

MCP 是"养飞轮"的手段，长期目标是 DINQ Hiring Agent（Mercor 非雇佣版）。两步是演化关系，不是平行业务。

②

Tier 1/2/3 是骨架

不是营销话术，是数据闭环必经路径。Tier 3 飞轮（Roast + 作品建档）决定 12 个月后 Tier 2 商业化能不能撑住。

③

多形态交付 = B 端溢价关键

80 万和 30 万年签的差别不是人数，是形态丰富度。六种形态能把 Strategic 卖到 200–400 万。
